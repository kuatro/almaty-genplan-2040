# Генплан Алматы 2040 — интерактивный оверлей

Растровый генеральный план г. Алматы, привязанный к реальным координатам и
наложенный на карту (OpenStreetMap / спутник Esri). Веб-вьюер на Leaflet.

**Live:** https://kuatro.github.io/almaty-genplan-2040/

## Что внутри репозитория

| Файл / папка | Назначение |
|---|---|
| `index.html` | Вьюер (Leaflet): подложка OSM/спутник, оверлей генплана, ползунок прозрачности |
| `tiles/` | XYZ-тайлы генплана (EPSG:3857, зум 9–16), сжаты pngquant |
| `.nojekyll` | Отключает Jekyll на GitHub Pages (иначе папки с тайлами могут не отдаваться) |

## Как это сделано

1. Растр извлечён из исходного PDF (один JPEG ~24800×26500 px).
2. Геопривязка по опорным точкам (GCP) → `gdalwarp` в EPSG:3857 (Web Mercator).
   Среднеквадратичная невязка привязки ≈ 18 м.
3. Нарезка на XYZ-тайлы (`gdal2tiles`), сжатие `pngquant`.

Исходный PDF, полноразмерный JPEG, GeoTIFF и инструменты привязки в репозиторий
не входят (см. `.gitignore`) — это локальные артефакты.
